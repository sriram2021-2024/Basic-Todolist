#  Basic To-Do List App

A simple and clean To-Do List application built using HTML, CSS, and JavaScript.  
This project allows users to add, delete, and mark tasks as completed with a strikethrough effect.

---

##  Features

- Add new tasks  
- Delete tasks  
- Mark tasks as completed (strikethrough)  
- Simple and responsive UI  
- Data stored in memory (no backend)

---

##  Project Structure
root/
│── todo.html
│── todo.css

- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla JS)**
